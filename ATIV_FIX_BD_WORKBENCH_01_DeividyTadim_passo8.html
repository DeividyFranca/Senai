Passo 8 : Salvar e Exportar


O oitavo passo é salvar e exportar dados do Workbench, primeiro você deve clicar no ícone exportar na barra de ferramentas. Escolha o formato de exportação desejado e selecione as tabelas e os dados a serem exportados. Clique em exportar para salvar o arquivo. Com esses passos iniciais, você deve estar pronto para começar a usar o Workbench para gerenciar seus bancos de dados MySQL.

Pressione o butão para prosseguir ou retornar.
