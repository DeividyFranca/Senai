Tag Class

A Tag class é utilizada para especificar uma classe em HTML.

Criação da Classe

Para criar uma classe, adicione o caracter ponto (.), logo após o nome da classe, adicione chaves ({}) e dentro delas adicione a estilização em CSS.

Tag Id

A Tag id é utilizada para especificar um id exclusivo para um elemento HTML.

Criação de Id

Para criar um id, adicione o caracter hash (#), logo após o nome do id, adicione chaves ({}) e dentro delas adicione a estilização em CSS.
